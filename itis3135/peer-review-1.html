Project Peer Review 1

  • Home page
    • The homepage on first impression looks very well made. The logo in the top left with the centered menu bar looks very clean and allows for efficient naviagation.
    • Below the menu, the skyline image that extends the width of the page with the business name and motto centered in the image looks professionally designed.
    • The spacing of the conents below the skyline image could use some work. There is not enough whitespace between the heading "Find a Psychologist" and the content below it.
  • About
    • The menu top margin is different from the homepage which should be changed.
    • The content located in the 3 sub-headings is broken up into a grid style which is a great design choice. However, like the home page, this page suffers from not having enough white space between the content and headings.
    • The footer bottom margin differs from the home page.
  • FAQ
    • The FAQ page menu is the same as the home page menu which is very good. Not sure what went wrong with the about menu.
    • The information is stored in an interactive dropdown system which is well implemented.
    • The footer formatting is consistent with the home page.
  • Professionals
    • Inconsistencies with header and footer margins.
    • Spacing issues but otherwise well designed.
  • Application
    • Inconsistencies with header and footer margins.
    • The size of the form looks a bit small and awkward on larger screens. Not sure if this is an issue; just an observation.
  • Payment
    • Inconsistencies with header and footer margins.
    • This page seems to have the appropriate content but needs some css work in regard to spacing.
  • Contact
    • Inconsistencies with header and footer margins.
    • Form has appropriate content but could be sized larger.

Justin has a great website so far. He implemented many professional design elements in the home and FAQ pages. I believe his focus from here should be directed at CSS. His menu and footer formatting changes from page to page when it should be consistent throughout. Furthermore, There is not enough whitespace separating his paragraphs from their respective headings. Overall, I believe he has a great website that can become better by focusing on the formatting aspect.
